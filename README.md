# A completely differentiable Ovation Prime model
<img src="https://github.com/jah-26603/DEEP_LEARNED_OVATION_PRIME_2010/blob/master/solarwind_predictions.gif" alt="Aurora" width="1000">


The current implementation thus far is the deep learning implementation of the current Ovation Prime 2010 model (with seasonal differences). This current implementation is differentiable, and thus,
is easily able to incorporate different datasets of the aurora for more accurate auroral forecasting.
